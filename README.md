# pen-wire-wrapping-tool

# 1st version
This works fine but if used close to other pins the end will get caught and cut the wire.
Making the hole a 45 degree angle helps some. Really need a way to keep the pin straight up and down.
Adding a heat shrink around the opening to keep the wire straight up and down helps. Also added a nickel plate inbetween to keep the wire from poking through.
The hole should be smaller and parralell to the main pin hole.
